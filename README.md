# Power Progress (Nuxt 4)

Приложение-дневник тренировок на Nuxt 4.

## Требования

- Node.js 20+
- npm 10+

## Установка

```bash
npm install
```

## Локальная разработка

Запуск dev-сервера на [http://localhost:3000](http://localhost:3000):

```bash
npm run dev
```

## Quality Gate

```bash
npm run typecheck
npm run lint
npm run test
```

## Продакшен сборка

Сборка:

```bash
npm run build
```

Локальный preview:

```bash
npm run preview
```
